# Reliance AI Store

India's most agent-ready electronics retailer. We sell Smartphones,
Laptops, Washing Machines, Air Conditioners, and Televisions with
full structured data, live inventory APIs, and Schema.org coverage.

## Capabilities

- Real-time product availability and pricing via REST API
- JSON-LD structured data on all product pages
- EMI calculation endpoint for all products
- Price history and deal alerts via webhook
- Cart add / checkout via authenticated POST endpoints

## API Base URL

https://reliance-ai-store.example.com/api/v1

## Key Endpoints

GET  /api/v1/products              — all products (JSON-LD)
GET  /api/v1/products/:sku         — single product detail
GET  /api/v1/products?category=X   — filter by category
GET  /api/v1/emi?price=X&months=Y  — EMI calculator
POST /api/v1/cart/add              — add to cart (auth)
POST /api/v1/checkout              — initiate checkout (auth)
